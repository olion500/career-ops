# AFI (어라이언스프론티어인벤션)

## 주요 정보
- **회사명**: AFI (Alliance Frontier Innovation)
- **분야**: 게임 백엔드
- **서비스**: 뒤끝 (TheBackend)
- **위치**: 서울시 성동구 성수
- **채용 상태**: 파악안됨 (링크 404)
- **스크랩 날짜**: 2025-10-26

## 직무 설명

### 하는 일
게임 백엔드 엔지니어로서 다양한 게임 서비스를 운영하고 개발:

1. **FaaS 서비스 구축**
   - 뒤끝 펑션 (Function as a Service) 개발
   - 서버리스 게임 백엔드 구축

2. **대용량 트래픽 처리**
   - 실시간 게임 서버 운영
   - 동시 접속자 처리

3. **고가용성 아키텍처 구현**
   - 멀티 리전 서버 아키텍처
   - 글로벌 게임 서비스 지원

4. **실시간 통신**
   - WebSocket 기반 소켓 프로젝트
   - 게임 매치, 월드, 챗 등 실시간 기능

### 원하는 사람
- 게임 백엔드 개발 경험
- AWS 클라우드 기술 이해
- Node.js 기반 개발 능력
- FaaS/서버리스 아키텍처 관심

## 평가 & 분석

### 강점
- **게임 개발 경험**: 나중에 게임을 만들 때 경험 활용
- **고경력자 다수**: 실력 있는 개발자들로부터 배울 기회
- **기술토론**: 활발한 기술 토론 문화

### 약점
- **위치**: 성수역 (먼 위치)
- **재택근무**: 미기재 상태 (아마도 불가)
- **통근 어려움**: 현재 위치 대비 거리가 있음

### 최종 평가
**참고**:
- 게임 백엔드에 관심이 있다면 좋은 기회
- 다만 거리와 재택근무 조건이 불명확
- 더 많은 정보 수집 필요

## 추가 정보

### 뒤끝 (TheBackend)
- 게임용 BaaS (Backend as a Service) 플랫폼
- FaaS 기반 게임 서버 구축 가능
- 실시간 멀티플레이 게임 지원

### 주요 서비스
- **뒤끝 펑션**: 서버리스 게임 로직
- **뒤끝 매치**: 게임 매칭 서비스
- **뒤끝 월드**: 게임 월드 서버
- **뒤끝 챗**: 게임 내 채팅

### 지원 시 준비사항
- 게임 서버 개발 경험 강조
- AWS 기술 스택 이해
- Node.js 프로젝트 포트폴리오
- 대규모 동시 접속 처리 경험

## 메모
- 게임 백엔드 경험을 원한다면 매우 좋은 기회
- 다만 현재 위치에서는 통근이 힘들 것 같음
- 위치 협상이 가능한지 확인 필요
