# 핵심 제품

## 1. Vrew (브루) - AI 영상 편집기 🎬

### 개요
음성인식을 포함한 각종 딥러닝 기술을 활용하여 영상 편집 시간을 획기적으로 단축시켜주는 신개념 영상 편집 프로그램

### 핵심 기능
- **자동 자막 생성**: 영상을 올리기만 하면 자동으로 자막 생성
- **음성-텍스트 변환**: 영상 내용을 텍스트로 자동 변환
- **텍스트 기반 편집**: 문서를 편집하듯 쉽고 간편하게 영상 제작
- **AI 목소리**: AI 음성 합성 기능

### 성과
- **월 결제액**: 9억원 (2024년 기준)
- **유료 구독자**: 1년간 3.5배 증가
- **글로벌 확장**: 글로벌 사용자 30% 이상

### 타겟 사용자
- 유튜버, 크리에이터
- 교육 콘텐츠 제작자
- 영상 편집이 필요한 일반 사용자

## 2. vFlat (브이플랫) - AI 모바일 스캐너 📱

### 개요
인공지능을 활용한 모바일 스캔 앱으로, **한국 점유율 1위**를 기록한 서비스

### 핵심 기능
- **자동 영역 인식**: AI가 스캔할 영역 자동 인식
- **책 곡면 펴기**: 책의 휘어진 부분을 깔끔하게 펴줌
- **13개 딥러닝 모델**: 고품질 스캔 결과 제공
- **OCR (문자 인식)**: 스캔한 이미지의 텍스트 추출

### 성과
- **시장 점유율**: 한국 1위
- **유료 구독자**: 1년간 6배 증가
- **타겟 시장**: 학생, 직장인

### 사용 사례
- 책 디지털화
- 문서 스캔 및 관리
- 학습 자료 정리

## 3. VOC STUDIO - AI 고객 문의 분석 툴 💼

### 개요
자연어 처리 기술을 활용해 고객의 문의를 분석하고 인사이트 도출을 돕는 B2B 솔루션

### 핵심 기능
- **문의 자동 요약**: 딥러닝 NLP로 문의 내용 요약
- **감정 분석**: 불만/이슈/칭찬 포인트 자동 추출
- **자연어 클러스터링**: 유사한 문의 자동 그룹화
- **인사이트 도출**: 고객 피드백 패턴 분석

### 타겟 고객
- 고객 서비스팀
- CS 운영팀
- 제품/서비스 기획팀
