# 업스테이지


# 회사

LLM, Document AI 전문 AI 기업. 네이버/카카오 AI 출신 창업. Solar LLM, Document AI 등 B2B AI 솔루션 제공. 완전 원격근무 (원격장비 500만원 지원). 평균연봉 5,466만원.

# 제품

## 핵심 제품
- **Solar LLM**: 자체 개발 대규모 언어 모델 (22B~31B 파라미터)
- **Document AI**: OCR + LLM 기반 문서 정보 추출 (99% 정확도)

## 제품 현황
- Solar Pro 2: Artificial Analysis '지능 지표' 전세계 12위, 기업 기준 8위
- Ko-Arena-Hard-Auto 벤치마크에서 GPT-4, Claude 3 견줄 성능
- 단일 GPU 실행 가능 (DUS 기술로 효율 최적화)
- 온프레미스/클라우드/API 다양한 배포 옵션

## 차별점
- DUS(Depth Up-Scaling): 7B → 22B 효율적 확장 기술
- 한국어 특화 + 다국어 지원
- 오픈소스 모델 (무료 API 액세스)
