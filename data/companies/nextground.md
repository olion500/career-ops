# 넥스트그라운드


# 회사

부동산 정보 플랫폼 '집품' 운영. 2021년 설립, 현재 10명 → 2025년 하반기 25명 규모로 확장 중인 초기 스타트업. 역삼역 5분 거리.

# 제품

## 핵심 제품
- **집품**: 실거주자 리뷰 기반 부동산 정보 플랫폼

## 제품 현황
- 20만+ 거주 후기 보유
- 월 트래픽 30% 이상 성장 (2년간 매월)
- App Store 긍정 평가 다수 ("실거주자 후기가 차별점")
- 경쟁사: 호갱노노, 다방, 직방 대비 **리뷰 특화**

## 차별점
- 전세사기 예방 기능 (보증금 신고, 전세대출 가능 분석)
- 층간소음, 치안 등 눈에 보이지 않는 정보 제공
- 빌라/다세대주택 리뷰도 제공 (타 플랫폼에 없음)

# 직무

## 업무
- 집품 서비스 백엔드 개발
- API 및 아키텍처 설계, 구현, 테스트
- 모니터링 통한 서버 안정성 및 성능 개선

## 자격요건
- Java Spring 2년+
- MySQL 확장성 있는 구조 설계
- JPA/Hibernate/Querydsl ORM 활용
- NoSQL(MongoDB, Redis) 경험
- 대용량 트래픽 처리 경험

## 우대사항

- 테스트 코드 작성
- 24/365 무중단 서비스 경험
